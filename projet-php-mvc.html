1) Présentation du projet

Ce projet répond à un besoin de gestion RH simple : centraliser les informations des employés et faciliter les opérations courantes dans une interface unique.

Sur le plan technique, la séparation Modèle / Vue / Contrôleur structure le code, facilite la maintenance et permet d'ajouter des fonctionnalités sans casser l'existant.

4) Organisation générale

Le projet repose sur un point d'entrée principal qui oriente l'utilisateur vers la bonne action selon la page demandée. Les accès sensibles sont protégés par la gestion de session.

5) Gestion des employés

La partie métier couvre les actions essentielles : créer, consulter, modifier et supprimer un employé. Des contrôles simples (comme l'unicité du matricule) permettent d'éviter les erreurs de saisie.

6) Fiabilité des données

Les informations utilisateurs sont vérifiées avant enregistrement pour renforcer la qualité des données et limiter les risques liés aux entrées incorrectes.

7) Expérience utilisateur

L'interface facilite l'usage quotidien grâce à une navigation simple, une recherche rapide dans la liste et des actions accessibles directement.

8) Points techniques impactants (à présenter)

  • Architecture MVC : séparation claire entre données, logique métier et interface.
  • Routage central : toutes les actions passent par index.php pour garder un flux cohérent.
  • Sécurité de base : contrôle de session sur les pages sensibles.
  • Fiabilité des données : contrôle d'unicité avant insertion des employés.
  • Ergonomie : recherche rapide dans la liste pour améliorer l'usage.

9) Résumé du flux

L'utilisateur déclenche une action, le contrôleur traite la demande, le modèle récupère ou met à jour les données, puis la vue affiche le résultat final.

10) Technologies utilisées

Cette partie technique s'appuie sur un stack cohérent : PHP pour la logique métier, MySQL pour la persistance des données, Bootstrap/CSS pour l'interface et JavaScript pour les interactions dynamiques (recherche instantanée, navigation, affichage).

11) Conclusion

Ce projet présente une application métier réaliste, avec un socle technique solide en PHP/MVC et base SQL. Il démontre ma capacité à concevoir un outil structuré, maintenable et adapté à un besoin professionnel.
