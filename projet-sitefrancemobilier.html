1) Présentation du projet

SiteFranceMobilier est un projet web de présentation et consultation de produits, pensé pour guider l'utilisateur dans le catalogue via des filtres simples.

Côté technique, le projet suit une architecture MVC avec un front controller PHP. Il inclut aussi un parcours d'abonnement complet à une lettre d'information (saisie, validation, e-mail de confirmation).

3) Menu et filtres dynamiques

Le menu latéral charge les catégories et les gammes depuis les modèles, puis envoie les critères au contrôleur de consultation.

4) Consultation des produits

Le contrôleur récupère les filtres et appelle les modèles pour afficher la liste correspondante dans la vue produits.

5) Gestion de l'abonnement (lettre d'information)

Le projet intègre un parcours d'abonnement simple : saisie de l'e-mail, enregistrement et validation. Cette partie apporte une logique métier concrète au-delà de la simple consultation du catalogue.

Elle met en évidence la gestion de la fiabilité des données avec contrôle de validité et gestion des erreurs.

7) Confirmation d'inscription

Lors de l'inscription à la lettre d'information, l'application envoie un e-mail de confirmation avec un lien de validation unique. Si l'envoi échoue, l'abonné est supprimé pour garder une base cohérente.

8) Points techniques clés

Le projet combine une base technique claire (routing, MVC, SQL) et des fonctionnalités utiles côté utilisateur.

  • Navigation simple avec filtrage catégorie/gamme.
  • Catalogue clair pour faciliter la consultation produit.
  • Inscription newsletter avec validation par e-mail.
  • Architecture structurée pour séparer affichage, logique et données.

L'ensemble démontre une organisation propre du code et un flux fonctionnel complet, du catalogue produit jusqu'à la validation d'abonnement.

9) Technologies utilisées

PHP MVC SQL HTML / CSS Bootstrap PHPMailer

10) Conclusion

Ce projet illustre ma capacité à concevoir une application web structurée, orientée utilisateur et appuyée sur des choix techniques cohérents en PHP/MVC. Il associe consultation produit et logique métier réelle.
