Site de Gestion du Personnel

Projet BTS SIO SLAM - Application web PHP avec architecture MVC, authentification et gestion complète des employés.

1) Présentation du projet

Ce projet a pour objectif de gérer les employés d'une entreprise via une application web. L'utilisateur peut se connecter, consulter les employés, filtrer par service, ajouter, modifier et supprimer des fiches.

L'application est structurée en MVC pour séparer la logique métier, l'affichage et la gestion des requêtes.

3) Architecture MVC et routage

Le fichier index.php agit comme routeur principal. Selon le paramètre page, il charge le contrôleur correspondant et vérifie la session utilisateur sur les pages protégées. 

$page = $_GET['page'] ?? 'login';

switch ($page) {
  case "listeEmployes":
    if (!$auth->isLoggedOn()) {
      header("Location: index.php?page=connexion");
      exit();
    }
    require_once "controleurs/C_consulterEmployes.php";
    $controleur = new C_consulterEmployes();
    $controleur->action_listeEmployes($_GET['service'] ?? null);
    break;
}

4) Gestion des employés (CRUD)

Contrôleur - ajout d'un employé

public function action_ajout($matricule, $nom, $prenom, $service)
{
  if (is_null($this->modeleEmploye->GetEmploye($matricule))) {
    $employe = $this->modeleEmploye->Ajouter($matricule, $nom, $prenom, $service);
    $this->data['leMessage'] = is_null($employe)
      ? "L'ajout a échoué."
      : $employe->GetNom()." ".$employe->GetPrenom()." a été ajouté.";
  } else {
    $this->data['leMessage'] = "Matricule déjà existant, ajout annulé.";
  }
  require_once "vues/v_message.php";
}

5) Affichage et expérience utilisateur

La vue liste affiche les employés dans un tableau Bootstrap avec des actions Modifier / Supprimer. Une recherche dynamique côté client permet de filtrer instantanément le tableau. 

document.getElementById('searchInput').addEventListener('keyup', function() {
  let filter = this.value.toLowerCase();
  let rows = document.querySelectorAll('#employeeTable tbody tr');
  rows.forEach(row => {
    let text = row.textContent.toLowerCase();
    row.style.display = text.includes(filter) ? '' : 'none';
  });
});

6) Sécurité et bonnes pratiques

  • Contrôle de session avant les pages sensibles.
  • Échappement SQL avec mysqli_real_escape_string.
  • Échappement HTML avec htmlspecialchars dans les vues.
  • Gestion des redirections après actions critiques (modifier/supprimer).
